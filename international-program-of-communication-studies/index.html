Ilmu Komunikasi Universitas Muhammadiyah Yogyakarta (UMY) memiliki tradisi panjang menawarkan ilmu tentang dimensi sosial, budaya, politik, dan sejarah Komunikasi dan Media. Program Studi S1 Ilmu Komunikasi berdiri sejak tahun 1996. Dalam usianya yang ke-22 tahun, Jurusan Ilmu Komunikasi Universitas Muhammadiyah Yogyakarta (UMY) mengalami banyak kemajuan dari tahun ke tahun.

Saat ini kami memiliki kurikulum baru untuk menjamin mahasiswa menguasai kompetensi komunikasi yang dibutuhkan dalam bekerja di bidang komunikasi. Selain itu, secara lebih konkrit misalnya, mahasiswa mendapatkan mata kuliah praktek untuk meningkatkan kemampuannya. Dan untuk menghadapi tantangan global, Program Studi Ilmu Komunikasi membuka program Internasional yang bernama International Program of Communication Studies (IP-COS), yang memulai proses pembelajaran pada tahun akademik 2020/2021.

Fokus dari IP-COS adalah pada studi Hubungan Masyarakat (PR). Program ini menawarkan mahasiswa kesempatan dalam pengembangan bahasa dan pengalaman belajar di bidang Hubungan Masyarakat (PR). Dalam proses pembelajarannya, IP-COS menggunakan pengantar Bahasa Inggris. 

Ingin mempelajari Program Studi lainnya?
Untuk menunjang masa depan Anda sebagai pribadi yang Unggul dan Islami, Muda Mendunia, Universitas Muhammadiyah Yogyakarta menawarkan banyak pilihan program studi. Kami memiliki 3 program Vokasi, 23 program Sarjana, 7 program Magister, 3 program Doktor dan 5 program pendidikan Profesi.
Cari berdasarkan Fakultas atau Program Studi untuk menemukan program pendidikan yang terbaik untuk anda.
